# 1. CSS Flexbox
## What is Flexbox?
Flexbox (Flexible Box Layout) is a one-dimensional layout system used to arrange items in rows or columns.
Before Flexbox, developers used:
* Float
* Positioning
* Tables
which were difficult to manage.
Flexbox made layouts much easier.

---
## Why Use Flexbox?
Suppose you want:
```text
Home   About   Contact   Login
```
all in one row.
Flexbox can do this very easily.

---
## Main Concepts
### Flex Container
The parent element.

### Flex Items
The child elements.

---
## Important Properties
### justify-content
Controls horizontal alignment.
Values:
* flex-start
* center
* flex-end
* space-between
* space-around
* space-evenly

---
### align-items
Controls vertical alignment.
Values:
* center
* flex-start
* flex-end
* stretch

---
### flex-direction
Controls direction.
Values:
* row
* column

---
## Why Important?
Almost every modern website uses Flexbox.

---
# 2. CSS Grid
## What is Grid?
Grid is a two-dimensional layout system.
Flexbox works in:
* Row OR Column
Grid works in:
* Rows AND Columns
simultaneously.

---
## Example

```text
Header
----------------

Sidebar | Content

----------------
Footer
```
Grid is perfect for such layouts.

---
## Advantages
✔ Powerful Layout System
✔ Easy Responsive Design
✔ Better than Float

---
## Most Important Properties
* grid-template-columns
* grid-template-rows
* gap
* grid-column
* grid-row

---
# 3. CSS Media Queries
## What are Media Queries?
Media Queries make websites responsive.
Responsive means:
Website automatically adjusts for:
* Mobile
* Tablet
* Laptop
* Desktop

---
## Why Important?
A website should look good on all screen sizes.

---
## Example
Desktop:
```text
4 products in a row
```
Mobile:
```text
1 product per row
```
Media Queries make this possible.

---
## Uses
✔ Responsive Design
✔ Mobile Friendly Websites
✔ Different Layouts for Different Devices

---
# 4. CSS Transitions
## What is Transition?
Transition creates smooth changes between states.
Without Transition:
```text
Button changes instantly
```
With Transition:
```text
Button changes smoothly
```
---
## Examples
* Button hover effects
* Color changes
* Size changes

---
## Benefits
Makes websites look professional.

# 5. CSS Transforms

Transform changes an element's shape, position, or size.

---

### Scale

### Skew

# 7. CSS Overlays
## What is Overlay?
An overlay is a layer placed over another element.
Example:
```text
Image
+
Dark Transparent Layer
+
Text
```
---
## Common Uses
### Hero Sections
Background image with text.

### Image Hover Effects
Show text when hovering.

### Popups
Dark background behind popup.

# 8. CSS Hamburger Menu
## What is Hamburger Menu?
A menu represented by:
```text
☰
```
three horizontal lines.

---
## Why Used?
Mobile screens have less space.
Instead of showing all menu items:
```text
Home About Services Contact
```
we show:

```text
☰
```
and open menu when clicked.

---
## Common Uses
✔ Mobile Websites
✔ Responsive Navigation
